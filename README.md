# SkateNotes

O **SkateNotes** foi desenvolvido para atender a uma necessidade específica de um amigo: substituir o processo manual de registro de manobras de skate, que antes era feito em um caderno. O objetivo do app é oferecer uma maneira mais organizada e eficiente de acompanhar a evolução no skate, permitindo classificar e registrar manobras de forma estruturada, bem como adicionar observações para manobras específicas, de modo que o usuário não vai esquecer de um detalhe que o ajudou a acertar a manobra, pois estará anotado.

## Funcionalidades

- **Organização das Manobras:** Separe as manobras que deseja aprender, aprimorar ou que já domina.
- **Registro do repertório de manobras:** Como o usuário terá as seção "na base", ele pode selecionar facilmente as manobras que deseja executar em sua linha em um campeonato, garantindo que não vai deixar de fora uma boa manobra que acabou esquecendo que tinha em seu repertório. 
- **Anotações Personalizadas:** Adicione observações para cada manobra, facilitando a consulta durante os treinos.
- **Acompanhamento Contínuo:** Revise e ajuste seu progresso sempre que necessário.

O **SkateNotes** é uma ferramenta simples, mas poderosa, para qualquer skatista que deseja evoluir com mais controle e estratégia.
